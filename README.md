# Todo-Backend

## Description
A backend REST-API to manage a todo app endpoints.

To check live running visit [REST-API](http://todo-app-backend-api.herokuapp.com/docs/) for more information.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
